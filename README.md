# Website Theme Picker 🎨

A web application that helps users quickly choose the right color and font combinations by previewing them inside real website layouts.

Instead of guessing which colors or fonts work together, users can browse pre-built themes and instantly see how they look in a realistic one-page website preview.

---

## 🚀 Live Concept

> “Pick a real-looking theme. Launch faster.”

This project focuses on reducing **decision fatigue** by showing complete UI compositions instead of isolated color palettes.

---

## ✨ Features

- 🎯 **Theme Gallery**
  - Browse multiple ready-made website themes
  - Categories: Business, Coffee Shop, SaaS, Luxury, Startup

- 🖥 **Real Website Preview**
  - Each theme opens in a modal
  - Includes:
    - Navigation bar
    - Hero banner (left-aligned text + CTA buttons)
    - Content cards section
    - Icon features section
    - Footer / info section

- 🎨 **60-30-10 Color Rule Applied**
  - 60% Primary (main background)
  - 30% Secondary (section background)
  - 10% Accent (buttons, highlights)

- 🔤 **Font Pairing Preview**
  - Heading font + body font shown in real context

- 📋 **Copy to Clipboard**
  - Copy:
    - Color hex values
    - Font names
  - Quick toast feedback

- 🧭 **Category Filtering**
  - Filter themes based on use case
